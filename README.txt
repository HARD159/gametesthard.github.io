Игра находиться в процессе разработки

Для запуска игры откройте файл GAME.html в браузере

2D (вид с верху) игра расчитана на двух игроков

управление персонажами происходит на одной клавиатуре 

Первый игрок (Красный)
Движение персонажа:
W - "Вперед"
A - "Влево"
S - "Назад"
D - "Вправо"
Q - "Поворот против часовой стрелки"
E - "Поворот по часовой стрелке"
Space - Выстрел
R - Перезарядить

Второй игрок (Синий)
Движение персонажа:
Num 8 - "Вперед"
Num 4 - "Влево"
Num 5 - "Назад"
Num 6 - "Вправо"
Num 7 - "Поворот против часовой стрелки"
Num 9 - "Поворот по часовой стрелке"
Num "+" - Выстрел
Num "-" - Перезарядить
